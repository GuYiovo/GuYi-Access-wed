API Reference

一个接口，
搞定所有场景。

统一的 JSON 响应结构，支持 GET / POST，参数按优先级独立读取，彻底杜绝参数污染。

1
卡密验证模式
提交 card_code +device_hash，返回到期时间与全部云变量。
2
公开变量模式
仅提交 app_key，无需卡密，自动返回 is_public=1 变量。私有变量安全隔离。
3
统一响应结构
所有响应通过 jsonOut() 统一输出 {code, msg, data} 结构。
